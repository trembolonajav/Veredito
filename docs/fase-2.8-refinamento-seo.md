## Objetivo

Tornar o produto publicável com qualidade profissional.

## Blocos

### SEO Técnico

- Meta tags dinâmicas por página (título, descrição, canonical)
- Open Graph tags para redes sociais
- JSON-LD structured data (Article, NewsArticle, BreadcrumbList)
- `robots.txt` configurado
- Sitemap XML dinâmico
- Breadcrumbs semânticos
- Alt text em todas as imagens
- Single `<h1>` por página
- Viewport responsivo

### Performance

- Lazy loading de imagens
- Code splitting por rota
- Otimização de bundle
- Compressão de assets
- Cache headers

### Operação Editorial

- Notas internas nos conteúdos
- Histórico de revisões (quem editou, quando)
- Checklist de publicação real
- Alertas de prazo de revisão

### Distribuição

- Newsletter com template final
- Compartilhamento social (botões + preview)
- RSS feed

### Fontes Monitoradas (futuro)

- Painel de fontes com feed
- Ingestão assistida de decisões
- Fila de triagem editorial
